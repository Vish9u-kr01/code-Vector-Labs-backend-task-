# Product Browser — Frontend

React + Vite + Tailwind CSS frontend for the Product Browser app.

See the [root README](../README.md) for full setup instructions, the API reference, and an explanation of why this app uses cursor-based pagination.

## Quick start

```bash
cp .env.example .env
npm install
npm run dev
```

Requires the backend (see `../backend`) running and reachable at the URL set in `.env` (`VITE_API_URL`, defaults to `http://localhost:4000`).
